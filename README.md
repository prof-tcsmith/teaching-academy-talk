# Measuring What AI Can't — Teaching Academy Talk

Presentation slides for the Muma Teaching Academy Summer Grant 2025–2026 talk by Dr. Timothy (Tim) Smith, University of South Florida.

**View the slides:** https://prof-tcsmith.github.io/teaching-academy-talk/

## About

A 10-minute talk on measuring AI-durable process skills in programming education, covering:

- The cognitive paradox of AI in education
- Six AI-durable process skills mapped to Zimmerman's SRL cycle
- The ForMetrix learning platform
- A validation study design using two established psychometric instruments
